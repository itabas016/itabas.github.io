Merging vs Rebasing

git rebase 和 git merge 都是用来合并分支，只不过方式不太相同。git rebase 经常被人认为是一种 Git 巫术，初学者应该避而远之。但如果使用得当，它能省去太多烦恼。在这篇文章中，我们会通过比较找到Git工作流中所有可以使用rebase的机会。
